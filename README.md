# UMAX 发型设计应用

UMAX是一款专注于发型设计的应用，通过AI技术分析用户脸型，推荐最适合的发型，并提供详细的理发指南。

## 主要功能

- **脸型分析**：上传自拍照片，系统分析脸型特征
- **发型推荐**：基于脸型分析结果推荐合适的发型
- **虚拟试戴**：查看推荐发型在自己脸上的效果
- **理发指南**：生成详细的理发步骤，方便与理发师沟通
- **分享功能**：将发型效果分享给朋友征求意见

## 技术栈

- React 18
- TypeScript
- Material UI
- React Router

## 安装与运行

1. 克隆仓库
   ```
   git clone <repository-url>
   ```

2. 安装依赖
   ```
   cd umax-hairdesign
   npm install
   ```

3. 启动开发服务器
   ```
   npm start
   ```

4. 构建生产版本
   ```
   npm run build
   ```

## 应用流程

1. 用户自拍上传（正面和侧面照片）
2. AI分析脸型，提供匹配度评分
3. 推荐适合的发型，并可以筛选
4. 虚拟试戴查看效果
5. 用户选择喜欢的发型，生成理发指南
6. 分享给朋友或理发师
